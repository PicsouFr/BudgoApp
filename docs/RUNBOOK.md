# RUNBOOK

## Objectif
Lancer l’environnement de dev en local:
- MySQL (Docker)
- API FastAPI
- Web PHP

## Prérequis
- Windows + Git Bash
- Docker Desktop (en cours d’exécution)
- Python 3.11+
- PHP 8.x
- (optionnel) DBeaver, Insomnia

## Arborescence utile
- services/api : API FastAPI
- apps/web : Web (PHP/HTML/CSS/JS)
- docs : documentation
